# snake-game-with-leap-motion

## Prerequisites

Leap Motion

```
https://developer.leapmotion.com/tracking-software-download
```

Python2 to use leap motion python API
```
for mac user
python2 -m pip install pyautogui==0.9.40 --user
```

## Deployment

python2 snake.py
